
## Problem Breakdown

```
User story 1 - Post pictures
----------------------------
As a user,
so that I can share what I am doing,
I want to be able to post pictures to instagram.
```

```
User story 2 - View all posts
-----------------------------
As a user,
so that I can see what others are doing,
I want to be able to see all posted pictures on instagram.
```

```
User story 3 - Like a picture
-----------------------------
As a user,
so that I can let others know I viewed a photo,
I want to be able to 'like' posts.
```

```
User story 4 - Sign up
----------------------
As a user,
so that I can have my own secure account,
I want to be able to sign up to instagram.
```

```
User story 5 - View individual post
-----------------------------------
As a user,
so that I can see a picture of an individual photo,
I want to be able to click on a post.
```

```
User story 6 - Delete post
--------------------------
As a user,
so that I can housekeep my content,
I want to be able to delete any photos that I no longer require.
```

```
User story 7 - Write comments
-----------------------------
As a user,
so that I can interact with other users and their content,
I want to be able to write comments on their posts.
```

```
User story 8 - See comments on timeline
---------------------------------------
As a user,
so that I can see how popular my content is,
I want to be able to see the number of comments on a photo.
```
