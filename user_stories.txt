User Stories:

As a user,
so that I can securely use the site,
I want to sign up.

As a user,
so that I can securely use the site.
I want to log out.

As a user,
so that I can securely use the site,
I want to sign in.

As a user,
so that I can share my experiences and art,
I want to post pictures.

As a user,
In case I change my mind,
I want to be able to edit and delete my pictures.

As a user,
so that my pictures are safe,
I want to be the only one who can edit or delete them.

As a user,
so I can interact with other users,
I want to write comments for their pictures.

As a user,
so that I can show my appreciation,
I want to like pictures.

As a user,
so I can draw pleasure from using the site,
I want it to be styled beautifully.
