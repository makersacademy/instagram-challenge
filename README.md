Installation
------------

1. Clone the repository

2. Run
```bash
bundle install
```

3. Run
```bash
rails server
```
... and enjoy the app at _localhost:3000_

Technologies Used
-----------------

* Ruby on Rails 4.*

* Postgresql

* RSpec and Capybara suites for testing

* Runs __Thin__ server in dev-env and __Puma__ in production
