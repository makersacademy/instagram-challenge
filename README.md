# Instagram Challenge

Ruby on rails v5.0.1

### System dependencies
 - Paperclip
 - Devise
 - Simple form

### Configuration
* fork/clone this repo
* cd into project folder
* run `bin/rake db:create` to create database
* run `bin/rails s` to start the server
* You can run the test suite with `rspec`
