## STACK

```
  Back-end:
  Ruby 2.5.0
  Rails 6.0.0
  Database: Postgres
```
```
  Front-end:
  HTML, CSS, Javascript, jQuery
  Bootstrap
```

How to use it:
---------------

``Clone this repo``


to run the app:

```
rails server
open http://localhost:3000
```

to run tests:

```
 bundle install
 rake db:setup
 rspec
```
