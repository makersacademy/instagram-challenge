## User Stories ##
```
As a User,
So I can enter Instagram,
I want to be able to Sign Up.
```
```
As a User,
So I can have my access to my account,
I want to be able to Sign In/Sign Out.
```
```
As a User,
So I can share my photos,
I'd like to be able to post a picture.
```
```
As a User,
So I can see all my images,
I'd like to have them all displayed on my profile.
```
```
As a User,
So I can see an image closer,
I'd like to be able to click and have the images full page.
```
```
As a User,
So I can share my opinion on other's images,
I'd like to be able to comment on other people's images.
```
