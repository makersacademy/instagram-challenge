Instagram Challenge
===================

Task
----
* Build Instagram using Rails.

Functionality
-------------
Users can:
 * post pictures
 * write comments on pictures
 * like a picture

Installation Instructions
-------------------------
```
$ git clone git@github.com:GeekG1rl/instagram-challenge.git
$ cd instagram-challenge
$ bunde install
$ rake db:setup
$ rspec
```

Run the app
-----------
```
$ rake db:seed
$ rails s
$ open http://localhost:3000
```
